# 🛡️ Админ Панель

## Обзор

Админ панель доступна по адресу `/admin` и предоставляет:

- Управление кошельками админов
- Список участников с ролями и XP
- Настройки клуба
- Выдачу бейджей

---

## Доступ

### Роли с доступом

| Роль | Доступ |
|------|--------|
| **owner** | Полный доступ, может управлять админами |
| **admin** | Доступ к управлению участниками |
| speaker/listener | Нет доступа |

### Тестовые кошельки

```
Owner: 0xOwnerWallet123456789
Admin: 0xAdminWallet987654321
```

---

## Вкладки

### 1️⃣ Кошельки (Wallets)

**Назначение:** Управление адресами с правами админа.

**Функции:**
- Просмотр Owner wallet
- Просмотр/редактирование Admin wallets
- Добавление новых админов

**API:**
```http
GET  /api/admin/settings
POST /api/admin/settings
```

**Интерфейс:**
```
┌────────────────────────────────────────┐
│ 👑 Owner Wallet                        │
│ 0xOwnerWallet123456789                 │
├────────────────────────────────────────┤
│ 🔐 Admin Wallets                        │
│ 1. 0xAdminWallet987654321    [✖]      │
│ 2. 0xNewAdmin...             [✖]      │
│                                        │
│ [+ Add Admin Wallet]                   │
└────────────────────────────────────────┘
```

---

### 2️⃣ Участники (Members)

**Назначение:** Список всех участников клуба.

**Информация об участнике:**
- Имя и аватар
- Роль (owner/admin/speaker/listener)
- XP и уровень
- Дней в клубе
- Telegram статус

**Действия:**
- Повысить до speaker
- Выдать бейдж
- Удалить из клуба

**API:**
```http
GET  /api/users
PUT  /api/users/{id}/role
POST /api/users/{id}/badges?badge_key=xxx
```

**Интерфейс:**
```
┌────────────────────────────────────────────────┐
│ Members (4)                                    │
├────────────────────────────────────────────────┤
│ ┌───┐                                            │
│ │👤│ Club Owner        👑 owner                │
│ └───┘ Level 5 • 10,000 XP • 30 days           │
│                                                │
│ ┌───┐                                            │
│ │👤│ Club Admin        🛠️ admin                │
│ └───┘ Level 4 • 5,000 XP • 20 days            │
│                                                │
│ ┌───┐                                            │
│ │👤│ Test Listener     👂 listener             │
│ └───┘ Level 1 • 100 XP • 5 days               │
└────────────────────────────────────────────────┘
```

---

### 3️⃣ Настройки клуба (Club Settings)

**Назначение:** Общие настройки клуба.

**Параметры:**
- Название клуба
- Описание
- Максимум участников
- Режим регистрации (open/invite-only)
- Hand Raise настройки

**API:**
```http
GET  /api/club/settings
PUT  /api/club/settings
```

## Бейджи

### Типы бейджей

**Participation (Участие):**
| Бейдж | Описание | Условие |
|--------|----------|--------|
| 🌟 Early Member | Ранний участник | Присоединиться в первые 30 дней |
| 🎤 First Speaker | Первое выступление | Выступить на сессии |
| 🎙️ 10 Sessions | 10 сессий | Посетить 10 live сессий |

**Contribution (Вклад):**
| Бейдж | Описание | Условие |
|--------|----------|--------|
| 💬 Active Commenter | Активный комментатор | 50+ комментариев |
| 👏 Supporter | Поддержка | 100+ лайков |

**Authority (Авторитет):**
| Бейдж | Описание | Условие |
|--------|----------|--------|
| ⭐ Core Member | Основной участник | 5000+ XP |
| 🌟 Host | Ведущий | Провести 5+ сессий |

### Выдача бейджа

**Автоматическая:** Бейджи выдаются автоматически при выполнении условий.

**Ручная:** Админ может выдать бейдж через API:

```bash
curl -X POST "https://api/users/user-001/badges?badge_key=early_member&admin_id=owner-001"
```
